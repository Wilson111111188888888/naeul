# 🔎 SEO naeul — stratégie & plan d'action

Objectif : être visible (puis n°1) sur le sujet **« peau grasse / K-beauty peau grasse »**.

## La vérité honnête sur le « n°1 »
Personne ne peut *garantir* la 1re place : Google classe selon ~200 facteurs, dont beaucoup **hors-site** (liens entrants, autorité du domaine, ancienneté, concurrence). naeul.com est un **domaine jeune** → ranker n°1 sur « peau grasse » (très concurrentiel) prend **des mois** + de l'autorité.
**La bonne stratégie** : viser d'abord les **requêtes longue traîne** (moins concurrentielles) qu'on PEUT gagner vite, accumuler du contenu + des liens, et grimper. Le on-page est notre force ; le hors-site, c'est le travail dans le temps.

## ✅ Ce qui est fait (on-page / technique — sous notre contrôle, optimisé)
- **Metadata** par page (title/description ciblés « peau grasse »), canonicals, OG/Twitter, `lang=fr`, `metadataBase`.
- **Schema.org** : Organization + WebSite (sitewide, avec réseaux), Product + FAQPage (produit), Article + BreadcrumbList (articles).
- **sitemap.xml** + **robots.txt** auto (toutes les pages + 12 articles).
- **12 articles** couvrant le cluster « peau grasse » : niacinamide, AHA/BHA, Centella, exosomes, microbiome, pores, routine, cercle vicieux, guide K-beauty…
- **Maillage interne** (articles ↔ produit ↔ FAQ), **alt-text** optimisés, images compressées, **perf** (Next, images priority).
- Site **mobile-first**, rapide, HTTPS.

## 🟡 Le plus gros levier ON-SITE restant = plus de contenu ciblé
Ajouter des articles sur des requêtes à **intention forte** non encore couvertes, ex. :
- « meilleur sérum peau grasse » / « sérum peau grasse avis »
- « peau qui brille en journée : que faire »
- « comment réduire le sébum naturellement »
- « niacinamide ou acide salicylique »
- « pores dilatés nez » / « peau grasse déshydratée »
→ **Dis-le moi, je les rédige** (1500-2000 mots, optimisés, maillés). C'est ce qui fait le plus avancer sur le site.

## 🔴 Ce qui dépend de TOI (hors-site = le vrai facteur de classement d'un domaine jeune)
1. **Google Search Console** (gratuit, indispensable) :
   - Vérifier le domaine → coller le token dans Vercel : variable `GOOGLE_SITE_VERIFICATION` (le code est déjà prêt à l'utiliser).
   - **Soumettre le sitemap** : `https://naeul.com/sitemap.xml`.
   - **Demander l'indexation** des pages clés (home, /le-produit, guide blog).
2. **Backlinks** (le nerf de la guerre) : être cité/lié depuis d'autres sites — blogs beauté, presse, annuaires beauté/clean, partenariats, marketplaces, liens depuis tes bios **Instagram/TikTok**, collaborations créatrices.
3. **Régularité** : publier du contenu régulièrement (1-2 articles/mois) → Google aime la fraîcheur.
4. **Signaux de marque** : recherches « naeul » qui augmentent (com' réseaux), avis, mentions.
5. **Avis clients réels** (après lancement) = contenu frais + EEAT (confiance).
6. **Bing Webmaster Tools** aussi (rapide, bonus).

## Suivi
- Après quelques semaines d'indexation : regarder dans Search Console les requêtes qui amènent des impressions → créer/renforcer le contenu sur celles-là (boucle vertueuse).

**Résumé** : le on-page est au maximum. Pour le n°1, il faut maintenant **du temps + des liens + du contenu régulier**. Je peux écrire autant d'articles que tu veux ; le reste (GSC, backlinks, réseaux) est entre tes mains.
